# Implementação e avaliação do algoritmo ACO para o problema do Caixeiro Viajante
---

Implementação do algoritmo Ant Colony Optimization (ACO) para resolução do Problema do Caixeiro Viajante (TSP) em grafos totalmente conexos com 5, 10, 20 e 50 cidades.
O projeto avalia o comportamento do algoritmo utilizando diferentes quantidades de cidades e pesos de arestas definidos pelos autores.
